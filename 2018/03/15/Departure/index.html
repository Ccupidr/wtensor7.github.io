发表于 2018-03-15

一开始刚接触CSDN的时候就特别喜欢，毕竟是把自己的东西传到网上，让有相似问题的能搜到，满满的成就感。但是还是觉得CSDN博客有点low，定制化的东西太少(也许是我自己技术不够)，一直都想要自己搭建一个博客，一开始尝试这搭建 wordpress博客，但是它显得太臃肿了，一些没用的太多了。之后在网上看一个个人博客的时候，hexo这个简单快捷的博客框架就出现在我的眼前。这个时候的我真的是小白，很多东西都不懂，就一路照着网上的不同的hexo教程胡乱安装，中间虽然有些小插曲吧，但是结果是好的，我搭建成功了，前前后后大概花费了我五个小时。
昨夜搞到2点，今天中午下课回来又搞到下午4点(2018/3/15 下午9:37:04)

在这里我贴一个CSDN博客的链接，写的很少，各位路过的大神轻喷，先介绍一下我目前的情况。 博主本人软工大二在读，至于在什么学校，就保密了，反正也不是那么好。以后决定把自己学的东西都一点一点记录下来，也算是自己成长过程的一个见证，说不定以后回忆起这段美好的大学时光呢！

using namespace std;
const int maxn = 1e5+5;
struct node{
int key;
int next;
}a[maxn];
vectorv1,v2;
int vis[maxn];
int main(int argc, char *argv[]){
int root,n;
cin>>root>>n;
for(int i=0;i<n;i++){
int x;
cin>>x;
cin>>a[x].key>>a[x].next;
}
for(int i=root;i!=-1;i = a[i].next){
if(vis[abs(a[i].key)]){
v2.push_back(i);
}
else {
v1.push_back(i);
vis[abs(a[i].key)] = 1;
}
}
int l1 = v1.size(),l2 = v2.size();
for(int i=0;i<l1-1;i++){
printf(“%05d %d %05d\n”,v1[i],a[v1[i]].key,v1[i+1]);
}
if(l1>=1)
printf(“%05d %d -1\n”,v1[l1-1],a[v1[l1-1]].key);
for(int i=0;i<l2-1;i++){
printf(“%05d %d %05d\n”,v2[i],a[v2[i]].key,v2[i+1]);
}
if(l2>=1)
printf(“%05d %d -1\n”,v2[l2-1],a[v2[l2-1]].key);
return 0;
}
